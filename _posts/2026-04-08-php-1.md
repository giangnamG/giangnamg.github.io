- `Include & Require` function
    - Ngoài include các file trên hệ thống, hàm `include` còn import file của 1 trang web
        
        ```php
        // static file in system
        include('vars.php');
        // file in external network
        include 'http://www.example.com/file.php?foo=1&bar=2';
        ```
        
    - `include` còn hỗ trợ đọc file qua quá schema khác như:
        - `file`
        - `schema php`: `php://filter/read=convert.base64-encode/resource=home.php`
            
            Giả sử khi ta truy cập file `flag.php` này:
            
            ```php
            <?php
            echo "Can you find out the flag?";
            //flag{f04a523c-3b10-4f3b-8a1c-db90bd2fc3d1}
            ?>
            ```
            
            Khi hiển thị html ở browser, chúng ta không thể nhìn thấy phần comment ở trong code php. Vậy nên để đọc được source code của php. Ta sẽ sử dụng `schema php` như sau
            
            `http://some-web.com/?file=php://filter/read=convert.base64-encode/resource=flag.php`
            
            Payload này sẽ trả về nội dung của file `flag.php` nhưng bị mã hóa base64, sau đó giải mã để đọc được source code
            
        - `ftp`
        - …
    - Cho phép đọc file sau dấu `?` - `Any version`
        
        ```python
        GET /?file=source.php?/../../../../../../flag
        ```
